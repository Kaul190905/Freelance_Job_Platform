
# Freelance Micro Job Platform

## Description
A platform where users post small development tasks and freelancers bid to complete them.

## Features
- JWT login
- Job posting and bidding
- Razorpay escrow payments
- Real-time messaging using WebSockets
- Reputation and rating system

## User Flow
Client: Login → Post Job → Receive Bids → Choose Freelancer → Pay → Release Payment  
Freelancer: Login → Browse Jobs → Bid → Complete Job → Receive Payment

## Tech Stack
Frontend: HTML, CSS, JS  
Backend: Your choice 
Realtime: Socket.io  
Database: Mongodb/MYSQL
